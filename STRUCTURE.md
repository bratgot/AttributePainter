│   │
│   │   Types.h                     ← shared POD types: Vec3f, Color3f, Ray, HitResult, enums

│   │   MeshSampler.h               ← KD-tree + BVH ray-cast + colour buffer
│   │   USDColorWriter.h            ← USD primvar read/write interface
│   │   UndoStack.h                 ← header-only: stroke-level undo/redo
│   │   ViewportBrushKnob.h         ← custom DDImage::Knob: GL overlay + mouse events
│   │   AttributePainterOp.h        ← main GeoOp: owns all subsystems
│
├── src/                            ← all translation units
│   │
│   │   plugin.cpp                  ← NDKDescribe() entry point, plugin registration
│   │   AttributePainterOp.cpp      ← GeoOp: knobs, geometry_engine, paint callbacks, USD commit
│   │   ViewportBrushKnob.cpp       ← GL brush circle, ray building, mouse → paint dispatch
│   │   MeshSampler.cpp             ← KD-tree build/query, BVH build/traverse, Möller–Trumbore
│   │   USDColorWriter.cpp          ← OpenUSD primvar read/write, staged commit batching
│   │   BrushSystem.cpp             ← stub TU (BrushSystem is header-only)
│   │   UndoStack.cpp               ← stub TU (UndoStack is header-only)
│
├── cmake/                          ← reusable CMake Find modules
│   │
│   │   FindNukeNDK.cmake           ← finds Nuke headers + DDImage.lib, creates Nuke::DDImage target
│   │   FindOpenUSD.cmake           ← finds USD headers + libs, creates USD::usd etc. targets
│
├── scripts/                        ← build + CI helper scripts
│   │
│   │   build.sh                    ← Linux/macOS one-shot build (gcc/clang, sanitizers, install)
│   │   build.bat                   ← Windows one-shot build (auto-finds MSVC via vswhere)
│   │   setup.ps1                   ← Windows: checks prerequisites, generates stubs, optional build
│   │   generate_stubs.sh           ← Linux/macOS: writes minimal NDK+USD header/lib stubs for CI
│   │   generate_stubs.ps1          ← Windows: PowerShell equivalent of generate_stubs.sh
│   │   run_clang_tidy.py           ← runs clang-tidy over src/, prints summary
│   │   package_release.sh          ← bundles build artefacts into versioned .tar.gz for GitHub Release
│   │   cppcheck_suppressions.txt   ← known-safe suppressions for cppcheck
